City Traverse — Temperature Analysis

This tool reads a combined CSV file from the temporary_data/ folder, applies temperature corrections, and produces three interactive outputs: a corrected-temperature vs. local-time scatter plot, a distance-vs-temperature plot with a Vapor Pressure Deficit (VPD) overlay, and a Folium map with color-coded temperature markers. Clicking a point on the distance plot pans the map to that location.

Place exactly one combined CSV file (with corrections) in the temporary_data/ folder before running.

Run Parameters

Adjust the options below, then click Run Analysis.

Parameter Description Default Your Value
--distanceflag How distance is calculated on the x-axis of the distance plot.
Mode 1: straight-line distance from the starting point.
Mode 2: cumulative distance traveled along the route. 		2 (cumulative)
--halfgraph When enabled, the x-axis of the distance plot is limited to the first half of the maximum distance, making it easier to compare outbound and return legs side by side. true (half range)
