Custom Mapping Parameters

Set parameters for the recursive subsampling and contouring script. Default values are pre-filled. Change any value you want, then click Run with These Parameters.

Parameter Description Default Your Value
--min-samples Minimum number of data points in a region before it is subdivided further. 10
--min-cell-samples Minimum number of data points required to keep a subregion cell. Cells below this are discarded. 4
--predicate Spatial join method used to assign points to polygons. intersects
--color-table Color scheme for the temperature map. 1 = blue-cyan-green-yellow-red. 2 = blue-green-red. 1
--show-borders Show rectangle borders on the static plots and contour image. Default is borders off. off (no borders)
--no-static-plots Whether to generate static Matplotlib plots (Static 1-4). Disable to skip and run faster. enabled

The primary output files are the .kmz file and the .html file, both below. You can view the .html file directly in your browser. The .kmz file is for Google Earth Pro, a free application you need to install on your computer. Download the .kmz file and just select it. That will automatically open Google Earth Pro and display the results.
